We don't actually need a theorem to answer this question, though: we can just test every option.

  1. \(\ds\diff{Q}{t}=0\text{,}\) so \(Q(t)=0=5\cdot 0 = 5\ds\diff{Q}{t}\text{,}\) so (a) is a solution.
  2. \(\ds\diff{Q}{t}=5e^t=Q(t)\text{,}\) so \(Q(t)=\ds\diff{Q}{t} \neq 5 \ds\diff{Q}{t}\text{,}\) so (b) is not a solution.
  3. \(\ds\diff{Q}{t}=5e^{5t}=5Q(t)\text{,}\) so \(Q(t)=\dfrac{1}{5}\ds\diff{Q}{t}\neq5\ds\diff{Q}{t}\text{,}\) so (c) is not a solution.
  4. \(\ds\diff{Q}{t}=\frac{1}{5}e^{t/5}=\frac{1}{5}Q(t)\text{,}\) so \(Q(t)=5\ds\diff{Q}{t}\text{,}\) so (d) is a solution.
  5. \(\ds\diff{Q}{t}=\frac{1}{5}e^{t/5}=\frac{1}{5}\left(Q(t)-1\right)\text{,}\) so \(Q(t)=5\ds\diff{Q}{t}+1\text{,}\) so (e) is not a solution.
