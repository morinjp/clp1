Solution 2.10.3.4.1
  • Solution 1: Using the chain rule, \(\ds\diff{}{x}\left\{\log(10x)\right\}=\dfrac{1}{10x}\cdot 10=\frac{1}{x}\text{.}\)
  • Solution 2: Simplifying, \(\ds\diff{}{x}\left\{\log(10x)\right\}=\ds\diff{}{x}\left\{\log(10)+\log x\right\} = 0+\frac{1}{x}=\frac{1}{x}\text{.}\)
