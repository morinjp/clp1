There is a new wrinkle in the next two examples. Each involves finding the minimum value of a function \(f(x)\) with \(x\) running over all real numbers, rather than just over a finite interval as in Corollary 3.5.13. Both in Example 3.5.18 and in Example 3.5.19 the function \(f(x)\) tends to \(+\infty\) as \(x\) tends to either \(+\infty\) or \(-\infty\text{.}\) So the minimum value of \(f(x)\) will be achieved for some finite value of \(x\text{,}\) which will be a local minimum as well as a global minimum.
