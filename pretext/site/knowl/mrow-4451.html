\begin{align*} \diff{}{x}\left\{\sin(2x)\right\}&=2\cos(2x)\\ \end{align*}

This is close to \(f'(x)\text{,}\) but we need to divide by 2.

\begin{align*} \diff{}{x}\left\{\frac{1}{2}\sin(2x)\right\}&=\cos(2x)\\ \end{align*}

So, \(f(x)=\dfrac{1}{2}\sin(2x)+C\) for some constant \(C\text{.}\) We can find \(C\) using the given information \(f(\pi)=\pi\text{.}\)

\begin{align*} \pi=f(\pi)&=\frac{1}{2}\sin(2\pi)+C\\ \pi&=C \end{align*}
