Now we can pick any \(Y\) that lies between \(f(a)\) and \(f(b)\) — see the middle figure above. The IVT  4  tells us that there must be some \(x\)-value that when plugged into the function gives us \(Y\text{.}\) That is, there is some \(c\) between \(a\) and \(b\) so that \(f(c) = Y\text{.}\) We can also interpret this graphically; the IVT tells us that the horizontal straight line \(y=Y\) must intersect the graph \(y=f(x)\) at some point \((c,Y)\) with \(a\le c\le b\text{.}\)
