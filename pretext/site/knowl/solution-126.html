Solution 1.6.4.20.1

The function is continuous for \(x\ne c\) since each of those two branches are polynomials. So, the only question is whether the function is continuous at \(x=c\text{;}\) for this we need

\begin{equation*} \lim_{x\to 2c^-}f(x)=f(2c)=\lim_{x\to 2c+}f(x). \end{equation*}

We compute

\begin{equation*} \lim_{x\to 2c^-}f(x)=\lim_{x\to 2c^-}6-cx = 6-2c^2; \end{equation*}
\begin{equation*} f(2c)=6-c\cdot 2c= 6-2c^2\text{ and} \end{equation*}
\begin{equation*} \lim_{x\to 2c^+}f(x)=\lim_{x\to 2c^+}x^2=4c^2. \end{equation*}

So, we need \(6-2c^2=4c^2\text{,}\) which yields \(c^2=1\text{,}\) i.e. \(c=-1\) or \(c=1\text{.}\)
