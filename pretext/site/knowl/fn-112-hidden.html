While it is clear that this satisfies the equation we want, it is less clear that it is the only function that works. To see this, assume that there are two functions \(f(t)\) and \(h(t)\) which both satisfy \(v'(t)=g\text{.}\) Then \(f'(t)=h'(t) = g\) and so \(f'(t)-h'(t) = 0\text{.}\) Equivalently \(\diff{}{t}\left( f(t)-h(t) \right) = 0.\) The only function whose derivative is zero everywhere is the constant function (see Section 2.13 and Theorem 2.13.12). Thus \(f(t)-h(t) = \text{constant}\text{.}\) So all the functions that satisfy \(v'(t)=g\) must be of the form \(gt + \text{constant}\text{.}\)
