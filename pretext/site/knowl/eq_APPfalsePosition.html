Equation C.3.1 false position method

Set \(c_n=\frac{a_n f(b_n) - b_n f(a_n) }{f(b_n)-f(a_n)}\text{.}\) If \(f(c_n)\) has the same sign as \(f(a_n)\text{,}\) then

\begin{equation*} I_{n+1}=[a_{n+1},b_{n+1}]\quad\text{with}\quad a_{n+1}=c_n,\ b_{n+1}=b_n \end{equation*}

and if \(f(c_n)\) and \(f(a_n)\) have opposite signs, then

\begin{equation*} I_{n+1}=[a_{n+1},b_{n+1}]\quad\text{with}\quad a_{n+1}=a_n,\ b_{n+1}=c_n \end{equation*}
