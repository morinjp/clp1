Solution 2.14.2.9.1

We differentiate using the chain rule.

\begin{align*} \diff{}{x}\{\log(5x^2-12)\}&=\frac{10x}{5x^2-12}\\ \end{align*}

Using the quotient rule:

\begin{align*} \ddiff{2}{}{x}\{\log(5x^2-12)\}&=\diff{}{x}\left\{\frac{10x}{5x^2-12}\right\}\\ &=\frac{(5x^2-12)(10)-10x(10x)}{(5x^2-12)^2}\\ &=\frac{-10(5x^2+12)}{(5x^2-12)^2} \end{align*}

Using the quotient rule one last time:

\begin{align*} \amp\ddiff{3}{}{x}\{\log(5x^2-12)\}=\diff{}{x}\left\{ \frac{-10(5x^2+12)}{(5x^2-12)^2} \right\}\\ &\hskip0.25in=\frac{(5x^2-12)^2(-10)(10x)+10(5x^2+12)(2)(5x^2-12)(10x)}{(5x^2-12)^4}\\ &\hskip0.25in=\frac{(5x^2-12)(-100x)+(200x)(5x^2+12)}{(5x^2-12)^3}\\ &\hskip0.25in=\frac{100x(-5x^2+12+10x^2+24)}{(5x^2-12)^3}\\ &\hskip0.25in=\frac{100x(5x^2+36)}{(5x^2-12)^3} \end{align*}
