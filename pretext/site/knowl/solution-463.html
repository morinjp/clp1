Solution 3.3.4.1.1

Using CorollaryÂ 3.3.8 (with \(A=20\) and \(K=5\)), solutions to the differential equation all have the form

\begin{equation*} T(t)=[T(0)-20]e^{5t}+20 \end{equation*}

for some constant \(T(0)\text{.}\) This fits (a) (with \(T(0)=20\)), (c) (with \(T(0)=21)\text{,}\) and (d) (with \(T(0)=40\)), but not (b) (since the constant has the wrong sign).

Instead of using the corollary, we can also just check each function for ourselves.

  1. \(\ds\diff{T}{t}=0=5\cdot0=5[T(t)-20]\text{,}\) so (a) gives a solution to the differential equation.
  2. \(\ds\diff{T}{t}=5[20e^{5t}]=5[T+20] \neq 5[T-20]\text{,}\) so (b) does not give a solution to the differential equation.
  3. \(\ds\diff{T}{t}=5[e^{5t}]=5[T-20]\text{,}\) so (c) gives a solution to the differential equation.
  4. \(\ds\diff{T}{t}=5[20e^{5t}]=5[T-20]\text{,}\) so (d) gives a solution to the differential equation.
