Square integers
\begin{align*} S &= \{n^2 | n \in \mathbb{Z}\}. \end{align*}
The set  1  \(S'= \set{n^2 | n \in \mathbb{N}}\) is not the same as \(S\) because \(S'\) does not contain the number \(0\text{,}\) which is definitely a square integer and \(0\) is in \(S\text{.}\) We could also write \(S = \set{n^2 | n \in \mathbb{Z}, n \geq 0}\) and \(S=\set{n^2 | n=0,1,2,\dots}\text{.}\)
