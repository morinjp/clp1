Definition 1.3.3 Informal definition of limit

We write

\begin{align*} \lim_{x \to a} f(x) &= L \end{align*}

if the value of the function \(f(x)\) is sure to be arbitrarily close to \(L\) whenever the value of \(x\) is close enough to \(a\text{,}\) without  3  being exactly \(a\text{.}\)
