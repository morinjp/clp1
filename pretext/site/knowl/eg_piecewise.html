Example 0.4.5 piece-wise function

Yet another numerical example.

\begin{align*} V:[-1,1] \to \mathbb{R} && \text{defined by } V(t) &= \begin{cases} 0 & \text{if } -1 \leq t \lt 0\\ 120 & \text{if } 0 \leq t \leq 1 \end{cases} \end{align*}

This is an example of a “piece-wise” function — that is, one that is not defined by a single formula, but instead defined piece-by-piece. This function has domain \([-1,1]\) and its range is \(\set{0,120}\text{.}\) We could interpret this function as measuring the voltage across a switch that is flipped on at time \(t=0\text{.}\)
