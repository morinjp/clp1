Solution 1.6.4.17.1

The function is continuous for \(x\ne c\) since each of those two branches are polynomials. So, the only question is whether the function is continuous at \(x=c\text{;}\) for this we need

\begin{equation*} \lim_{x\to c^-}f(x)=f(c)=\lim_{x\to c+}f(x). \end{equation*}

We compute

\begin{equation*} \lim_{x\to c^-}f(x)=\lim_{x\to c^-}8-cx = 8-c^2; \end{equation*}
\begin{equation*} f(c)=8-c\cdot c= 8-c^2\text{ and} \end{equation*}
\begin{equation*} \lim_{x\to c^+}f(x)=\lim_{x\to c^+}x^2=c^2. \end{equation*}

So, we need \(8-c^2=c^2\text{,}\) which yields \(c^2=4\text{,}\) i.e. \(c=-2\) or \(c=2\text{.}\)
